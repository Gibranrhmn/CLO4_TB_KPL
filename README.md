# CLO4_TB_KPL

### Penjelasan singkat mengenai aplikasi

Aplikasi ini dibuat untuk memenuhi tugas besar mata kuliah konstruksi perangkat lunak. Quiz Master adalah aplikasi kuis interaktif berbasis web yang memungkinkan pengguna.

1. Memilih kategori dan tingkat kesulitan
2. Menjawab soal dengan waktu terbatas
3. Menggunakan fitur hint & skip
4. Mendapatkan skor akhir & statistik performa
